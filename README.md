# 🔗 Lead Tracker Web App

A simple and powerful **Lead Tracker Application** built using HTML, CSS, and JavaScript. This app helps users save, manage, and track useful links (leads) efficiently.

---

## 🚀 Live Demo

🔗 https://apelabrazil29.github.io/Chrome-Extension/

---

## 📌 Features

* ➕ Save custom input links
* 🌐 Save current browser tab (Chrome feature)
* 🗑️ Delete all saved leads (double click)
* 💾 Persistent storage using Local Storage
* 🔗 Clickable links (open in new tab)
* ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Local Storage API
* Chrome Tabs API

## ⚙️ How It Works

1. Enter a URL in the input field
2. Click **SAVE INPUT** to store it
3. Click **SAVE TAB** to save current browser tab
4. Click **DELETE ALL** (double click) to clear all data
5. Saved leads are displayed as clickable links

---

## ⚠️ Important Note

* The **"Save Tab" feature works only in Chrome Extensions**
* It will NOT work on normal websites due to browser security restrictions
